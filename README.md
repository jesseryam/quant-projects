# Quant Projects

Pre-college quantitative finance projects built in Processing.

## gbm-basic
Simple Geometric Brownian Motion stock price simulator with 
Monte Carlo (10,000 simulations).

## gbm-jump-diffusion-regime-switching
Upgraded simulator adding:
- Merton (1976) jump diffusion for quarterly earnings shocks
- Volatility regime switching between low and high volatility states
- Visual color coding of regimes and earnings days on the price path

Both projects require Processing to run: https://processing.org
